# Assets para Cidade de Deus

Para adicionar suas próprias imagens:

1. banner-cdd.jpg (recomendado: 1920x600)
2. logo-cdd.svg (logo do projeto)
3. mapa-base.png (mapa de referência)
4. historias/ (fotos para as histórias)
5. midia/ (thumbnails para vídeos)

Formato recomendado:
- JPG para fotos
- SVG para logos
- PNG para transparência
