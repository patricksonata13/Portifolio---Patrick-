Instruções para atualizar no GitHub:

  1. Faça o upload da imagem usando o botão acima para visualizar
  2. Acesse seu GitHub e vá até o repositório do seu portfólio
  3. Navegue até a pasta "images/works/"
  4. Clique em "Add file" → "Upload files"
  5. Arraste a nova imagem ou clique para selecionar
  6. Certifique-se de que o nome do arquivo é "fotodeperfil.jpg"
  7. Clique em "Commit changes" para salvar
