Sobre Mim

Patrick Sonata é um roteirista e comediante carioca de 36 anos, nascido e criado na Cidade de Deus. Sua trajetória no humor para televisão começou em 2018, quando foi selecionado para a oficina de novos roteiristas da Globo.

Com experiência como ator e comediante nos principais palcos cariocas, Patrick também se dedica à pesquisa em linguagens populares e humor.

Como Resolver o Problema das Imagens

Passo a passo para criar a estrutura de pastas no GitHub

1

Acesse seu repositório no GitHub: Portifolio---Patrick-

2

Clique no botão "Add file" e selecione "Create new file"

3

Para criar a pasta public/images/creditos/, digite no campo do nome do arquivo:

public/images/creditos/.gitkeep

O arquivo .gitkeep é apenas um placeholder para forçar o GitHub a criar as pastas.

4

Role para baixo e clique em "Commit new file"

5

Agora faça upload das suas imagens para a pasta public/images/creditos/:

Clique em "Add file" → "Upload files"

Arraste as imagens para a área de upload ou selecione-as manualmente.

Certifique-se de que os nomes dos arquivos correspondem aos usados no código:

pabloeluisao.jpg
praprovar.jpg
pelada.jpg
doccidadecorreria.jpg
cidadecorreriapeca.jpg
humornegro.jpg
6

Após o upload, seu portfólio deve exibir as imagens corretamente.
