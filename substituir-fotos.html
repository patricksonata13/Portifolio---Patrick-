Substituir Fotos do Portfólio

Ferramenta para atualizar imagens do portfólio de Patrick Sonata

Como substituir as imagens:

  1. Para cada imagem abaixo, você pode:
    • Inserir uma URL de imagem da internet (ex: Google Drive, Imgur, etc.)
    • Ou fazer upload de uma imagem do seu computador
  2. Após inserir todas as imagens, role até a parte inferior da página
  3. Copie o código gerado e substitua as tags de imagem no seu código HTML original
  4. Para imagens do Google Drive: compartilhe a imagem como "Qualquer pessoa com o link pode visualizar", depois copie o link e use neste gerador
